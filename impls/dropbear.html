Supported protocols

Ciphers

aes128-ctr, 3des-ctr, aes256-ctr, aes128-cbc, 3des-cbc, aes256-cbc, twofish256-cbc, twofish-cbc, twofish128-cbc, blowfish-cbc, none,

Compression formats

zlib, zlib@openssh.com, none,

Hostkey formats

ecdsa-sha2-nistp256, ecdsa-sha2-nistp384, ecdsa-sha2-nistp521, ssh-rsa, ssh-dss,

Key exchange protocols

curve25519-sha256@libssh.org, ecdh-sha2-nistp521, ecdh-sha2-nistp384, ecdh-sha2-nistp256, diffie-hellman-group1-sha1, diffie-hellman-group14-sha1,

Message authentication codes (MACs)

hmac-sha2-256, hmac-sha2-512, hmac-sha1-96, hmac-sha1, hmac-md5, none,

User authentication methods

publickey, password, keyboard-interactive, none,
