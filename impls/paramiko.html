Remarks

  • Python library.
  • Built on PyCrypto, a Python C extension for low level cryptography.

Supported protocols

Ciphers

aes128-ctr, aes192-ctr, aes256-ctr, aes128-cbc, blowfish-cbc, aes192-cbc, aes256-cbc, 3des-cbc, arcfour128, arcfour256

Compression formats

zlib@openssh.com, zlib, none

Hostkey formats

ssh-rsa, ssh-dss, ecdsa-sha2-nistp256

Key exchange protocols

diffie-hellman-group1-sha1, diffie-hellman-group14-sha1, diffie-hellman-group-exchange-sha1, diffie-hellman-group-exchange-sha256, gss-group1-sha1-, gss-group14-sha1-, gss-gex-sha1-*

Message authentication codes (MACs)

hmac-sha2-256, hmac-sha2-512, hmac-md5, hmac-sha1-96, hmac-md5-96, hmac-sha1

User authentication methods

publickey, password, keyboard-interactive, gssapi-with-mic, gssapi-keyex
